# Local audio (not in Git)

Background music is served in dev from this folder (see `apps/client/vite.config.ts`).

Place this file here (do **not** commit it — GitHub rejects files over 100 MB):

`Valley of Olympus - Ethereal Ancient Greek Mythology Ambient Music - Athena IV (128k).wav`

The game loads it at `/audio/bg-music.wav` when you run `npm run dev`.
