# Sandbox Strategy Bake-Off

When Kalshi demo + ProphetX sandbox are both live, run one strategy per day (~$100 per venue):

| Day | Strategy | Metrics |
|-----|----------|---------|
| 1 | live_only | fill rate, leg failures, net P&L |
| 2 | static_only | capital lock time, opportunities/hour |
| 3 | combined | best overall score |

Set `TYCHE_STRATEGY` accordingly each day. Compare results before production.
