# Olympus Opp UI — Product Direction

Distilled from the brainstorming + mockup session.

## Core concept

A gamified command center for AI Opps (agents), themed as the Greek pantheon on
Mount Olympus. It should feel and play like a real 2D game (Stardew Valley /
Terraria), not a SaaS dashboard. Eventually each Opp runs its own profit-making
tasks (Etsy stores, TikTok pages, etc.) with an overseer coordinating the rest.

## Confirmed preferences

- **Full-screen map.** The Mount Olympus overworld fills the screen.
- **HUD as overlays.** Mission HUD, currency, and hotbar float over the map.
- **No persistent chat panel.** Chat opens only when you interact with an Opp.
- **Interact model.** Walk near an Opp/building and press `E` (game-style).
- **Centered interaction dialog.** Opening an Opp shows a centered overlay (not
  full-screen) with a small animation of the Opp, status info, and chat at the
  bottom.
- **Speech bubbles.** Opps show in-world speech bubbles tied to what they do.
- **Living world.** Opps move and act continuously, reflecting real task state.
- **Your character.** The user has a controllable avatar that walks around.
- **Smooth zoom.** Game-like mouse/pinch zoom, no zoom slider/bar in the UI.
- **Currency = Drachmas.** Gold coins + earning animations, never dollars.
- **Roster name = "Hall of Allies."**
- **Per-agent rooms/temples** integrated into one shared map (Stardew houses).
  Walk to a temple, press `E` to enter its interior, look around, press `E` to
  interact with the Opp inside.

## First map content

- Mount Olympus overworld with lots of empty space for growth.
- Main Headquarters (the player's command hall).
- A grand Temple of Zeus.
- A Zeus character.

## Future (not in first version)

- Real autonomous agents doing real tasks without per-action approval.
- Etsy/TikTok and other profit automations.
- Microphone input to talk to Opps.
- Distinct text-to-speech voices per Opp.
- Many specialized Opps + their temples, plus markets/shops/business zones.
- An ABCII dashboard ("Oracle") with deeper analytics.
